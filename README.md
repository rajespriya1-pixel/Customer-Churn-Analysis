# Customer-Churn-Analysis
# Objective:
To analyze customer churn data and identify key factors influencing churn behavior for business decision-making.

# Tools Used: MySQL

# Data Modeling:
Structured relational dataset with customer demographics, usage, support, and churn status.

# Key Metrics:
Churn rate, customer tenure, CSAT score, support tickets, contract type distribution.

# Key Insights:
Higher churn observed in low CSAT; longer tenure customers show lower churn.

# Conclusion:
Support ticket analysis shows consistent churn rates across ticket levels, indicating stable support handling; however, additional metrics like resolution time and customer satisfaction scores are required to better evaluate support efficiency and its impact on churn.
